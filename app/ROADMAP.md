# Google Tech News (GTN) - Roadmap 2026

## v1.1: The Intelligence Update
-   [ ] **AI News Summaries**: Use Gemini Nano (on-device) to generate 3-bullet summaries for every article.
-   [ ] **Smart Tags**: Auto-classify articles using ML into "Hardware", "Software", "AI", "Cloud".
-   [ ] **Voiceover**: Text-to-Speech playback for news articles while driving.

## v1.2: The Ecosystem Expansion
-   [ ] **Desktop Support**: Fully optimized Windows/macOS layout with side navigation.
-   [ ] **Chrome Extension**: "Save to GTN" button for desktop browsers.
-   [ ] **Wear OS Companion**: Quick headlines on the wrist.

## v2.0: Community
-   [ ] **User Accounts**: Sync bookmarks across devices.
-   [ ] **Comments**: GitHub-backed discussion threads per article.
-   [ ] **Contributor Mode**: Submit community articles via Pull Request.
